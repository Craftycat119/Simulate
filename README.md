# Симулятор Жизни — веб-версия (PWA)

Это версия игры для телефона (iPhone и Android), которая работает как обычный сайт
в браузере и может быть добавлена на домашний экран как отдельное приложение —
без App Store, Google Play, Xcode или Android Studio.

## Что реализовано
- Полный игровой цикл: работа (50 профессий, опыт, чаевые), образование (14 курсов,
  можно учиться на нескольких одновременно), семья (15 членов/статусов),
  здоровье/настроение/сытость с действиями для их восстановления
- Инвестиции: 50 акций, 10 криптовалют, 100 объектов недвижимости с улучшениями
- Казино: слоты (с анимацией), рулетка, блэкджек
- Собственность: машины/яхты/самолёты/апартаменты
- Достижения (27 шт.), журнал событий, случайные события
- Сохранение прогресса прямо в браузере телефона (localStorage) — данные не уходят
  на сервер, всё работает офлайн после первого открытия
- Кастомизация внешности персонажа

## Что НЕ перенесено в этой версии (можно добавить по запросу)
- Вкладка "Ставки" (спортивные события) из десктопной версии
- Звуковые эффекты
- Более детальная отрисовка аватара, чем в tkinter-версии (упрощена под Canvas)

## Как установить на телефон

Т.к. это статический сайт, ему нужен любой веб-хостинг (просто открыть index.html
двойным кликом с телефона не сработает — переходы по file:// блокируются
современными браузерами и Service Worker не подключится). Варианты, от простого
к сложному:

### Вариант А — самый простой: GitHub Pages (бесплатно, 5 минут)
1. Создайте новый репозиторий на github.com (можно приватный).
2. Загрузите в него содержимое папки `pwa/` (все файлы и подпапки).
3. В настройках репозитория → Pages → Source → выберите ветку `main` и папку `/root`.
4. Через минуту сайт будет доступен по ссылке вида
   `https://ваш-логин.github.io/имя-репозитория/`.
5. Откройте эту ссылку в Safari (iPhone) или Chrome (Android) с телефона.

### Вариант Б — Netlify Drop (тоже бесплатно, без регистрации)
1. Откройте app.netlify.com/drop с компьютера.
2. Перетащите папку `pwa/` в браузер.
3. Получите ссылку и откройте её на телефоне.

### Вариант В — свой сервер / хостинг
Просто скопируйте содержимое папки `pwa/` в любую папку, отдаваемую по HTTPS
(Nginx, Apache, Vercel, Cloudflare Pages и т.д.). HTTPS обязателен для установки
как приложение — по обычному http:// Service Worker работать не будет
(единственное исключение — localhost при локальном тестировании).

## Установка как приложение (после того как сайт открыт в браузере)

**iPhone (Safari):** кнопка "Поделиться" (квадрат со стрелкой) → «На экран
«Домой»» → Добавить.

**Android (Chrome):** меню (три точки) → «Добавить на главный экран» /
«Установить приложение».

После этого иконка появится на рабочем столе и будет открываться как обычное
приложение, на весь экран, без адресной строки браузера.

## Локальная проверка на компьютере
Из папки `pwa/` выполните:
```
python3 -m http.server 8000
```
и откройте `http://localhost:8000` в браузере.
